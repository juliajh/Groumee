# Groumee

약속 시간 조율을 도와주는 서비스 🕐<br>

평소엔 개인 웹캘린더로 쓰다가, 약속을 잡을 친구들끼리 그룹을 만들면 그룹 내 인원들의 개인 캘린더를 종합해 일정이 비는 날을 쉽게 확인할 수 있고 약속을 잡을 수 있습니다.
